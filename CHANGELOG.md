# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-23

### Added
- Initial release
- Dependency scanning for Node.js, Python, Go, Rust, Ruby, Java, Docker
- CVE detection via OSV.dev API
- EOL status checking via endoflife.date API
- A-F grading system for dependency health
- Markdown report output
- JSON output for CI/CD integration
- GitHub Action integration (`Releaserun/releaserun-action@v1`)
